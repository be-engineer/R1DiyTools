首先感谢 @rush ，没有他的付出，斐讯R1 应该不会这么快出现 root


其次，要 root 斐讯R1，usb焊接必不可少，TTL也顺道就焊上了吧。

思路：通过刷入 magisk 修改过后 boot 分区，实现不修改 system root（R1应该没有机会再升级了）
--------2019.9.5 更新：--------------

有同学说不能下载189的，再附上一个百度的：百度链接: https://pan.baidu.com/s/1v1I1CDJ-osW8MtUTWF3FgQ 提取码: 78um

建议食用方法（以下都是在获取到 root 权限之后的操作，获取root权限看更下面的说明）：
1，挂载 /system 分区，把斐讯的软件都干掉，命令如下：
  1. busybox mount -o remount,rw /system
  2. mkdir /system/app-disabled
  3. mv /system/app/* /system/app-disabled/
  4. reboot
复制代码


2，网络配置文件为 /data/misc/wifi/wpa_supplicant.conf，在文件后面添加如下内容(注意替换自己的 ssid 和密码)，即可联网：（帖子里面输入的命令行被吃掉了）
  1. network={
  2.         ssid="your_ap_ssid"
  3.         psk="your_ap_password"
  4.         group=CCMP TKIP
  5.         auth_alg=OPEN
  6.         priority=215
  7. }
复制代码

3，启动CarLife 命令为(怎么让app随系统启动，不需要手动干预，有知道的同学请不吝指教)：
  1. am start com.baidu.carlife/.CarlifeActivity
复制代码

4，MagiskManager在给 shell 赋权之后，就可以停用了，这个不停用也会占用系统资源，停用命令：
  1. pm disable com.topjohnwu.magisk
复制代码

----------以下为原贴---------

需要的工具：
rkflashtool
MagiskManager
投屏软件：AndroidScreencast

步骤：
1，下载附件(R1_rooted_boot.img)
2，斐讯重启到 rkusb 模式（命令： adb reboot bootloader ）
3，Linux下，用 rkflashtool 命令刷入修改后的 boot.img (命令: rkflashtool w boot < R1_rooted_boot.img），windows下有相应工具，请自行查找
4，重启R1（命令： rkflashtool b）
5，安装MagiskManager（adb install MagiskManager_7.3.2.apk)
6，打开投屏软件 (java -jar androidscreencast-0.0.13s-executable.jar) 或者任何其它自己熟悉的投屏软件
7，运行 adb shell su，在投屏软件上点击许可，至此可以得到完整的 root 权限

附件下载链接： https://cloud.189.cn/t/eyEFnmAvm6Bb（访问码：4557）

carlife 可用，默认情况下是卡出翔，不过把斐讯家的几个软件禁用之后，反应虽然不是很让人满意，但总算是有反应了。
有一点点希望了。
禁用方式: 进入 adb shell su 后：
1， 查看phicomm 家的软件: pm list packages |grep phicomm
2, 把斐讯家的软件全部（除launcher外，毕竟还需要用来启动carlife）禁用:
  1. pm disable com.phicomm.speaker.airskill
  2. pm disable com.phicomm.speaker.productiontest
  3. pm disable com.phicomm.speaker.bugreport
  4. pm disable com.phicomm.speaker.otaservice
  5. pm disable com.phicomm.speaker.device
  6. pm disable com.phicomm.speaker.ijetty
  7. pm disable com.phicomm.speaker.systemtool
  8. pm disable com.phicomm.speaker.netctl
  9. pm disable com.phicomm.speaker.player
  10. pm disable com.phicomm.speaker.exceptionreporter
