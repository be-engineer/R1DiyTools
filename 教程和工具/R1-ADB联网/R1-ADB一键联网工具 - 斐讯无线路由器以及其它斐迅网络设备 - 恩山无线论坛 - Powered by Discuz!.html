正常所需命令如下
  1. adb connect 192.168.43.1（R1配网发出的wifi信号）
  2. adb shell settings put secure install_non_market_apps 1
  3. adb push adb-join-wifi_1.0_fix.apk /data/local/tmp/
  4. adb shell /system/bin/pm install -t /data/local/tmp/adb-join-wifi_1.0_fix.apk
  5. adb shell am start -n com.steinwurf.adbjoinwifi/.MainActivity --esn connect -e ssid SSID -e password_type WPA -e password PASSWORD
